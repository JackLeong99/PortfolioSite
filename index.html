Server Boi

A discord bot made in python, designed to remotely launch and send commands to a minecraft server

More...

This was a passion project born from both interest and necessity. After deciding to host a Minecraft server for some friends on some old hardware, I realised that they wouldn't be able play whenever they want unless I were to leave it on 24/7 (costly) or manually boot it whenever they wanted to play (annoying). Thus I created this bot so they could control the server via discord.
